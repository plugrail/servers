# Security policy

## Reporting a vulnerability

Please report suspected security vulnerabilities privately to [security@plugrail.dev](mailto:security@plugrail.dev). Do not include exploit details in a public issue.

Please provide a clear description, steps to reproduce, potential impact, and any suggested mitigation. We aim to acknowledge reports within five business days and will coordinate disclosure after assessing the issue.

## Scope

This policy applies to the code in this repository and the jp-calendar service.
