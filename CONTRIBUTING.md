# Contributing to @plugrail/jp-calendar

Thank you for contributing. Please use GitHub Issues for bug reports and feature
requests. Include clear reproduction steps, expected behavior, and actual
behavior when reporting a bug.

For pull requests, keep each change focused, explain its purpose, and add or
update tests when behavior changes. Make sure the checks pass before requesting
review.

## Development setup

Install dependencies from the repository root, then run the package tests:

```sh
pnpm install
pnpm --filter @plugrail/jp-calendar test
```

Code must follow the repository's Biome formatting and linting rules.
